Step 1: Install necessary libraries automatically 
a) Run these two commands in console

pip install pipreqs
pipreqs

pipreqs will generate a file named "requirements.txt"

b) Run command below to install required libraries
pip install -r requirements.txt

Step 2: Launch guided tool, train, and perform real time test
a) Import preset or add gestures manually
b) Record data for each gesture
c) Train model after all data are captured
d) Proceed to real time test after model is ready

Note: Build-in help guide, message box, and video "train", "train_finger" are helpful.
Codes are available in github https://github.com/sqihang8/machine_learning_guided_tool.git
